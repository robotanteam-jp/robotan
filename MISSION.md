# Mission機能仕様 v2.1

## 概要

Mission は

**ロボタンが今、一緒に進める小さな任務**

を表示してください。

---

# コンセプト

Mission は

**「次の一歩」**

を表示します。

ロボタンは人生の目標を示すAIではなく、

「今できること」

を一緒に進めるサポートロボットです。

Mission は5〜15分程度で終わる小さな行動を表示します。

---

# Mission State

```ts
interface Mission {
  title: string;
  completed: boolean;
  tags: string[];
}
```

# Mission Complete演出

Missionを完了したら、以下の順番で処理してください。

## ① Mission Complete

Mission欄に

```
MISSION COMPLETE
```

を約1〜2秒表示する。

---

## ② 入力欄を一時ロック

Mission Complete表示中（約1〜2秒）は、

チャット入力欄と送信ボタンを一時的に無効化してください。

これはロボタンが次の任務を考えている演出です。

Mission更新後、自動で入力可能に戻してください。

---

## ③ 胸ランプ演出

# Mission Complete時の胸ランプ演出

胸ランプの色は Status を表しています。

そのため、Mission Completeでランプの色は変更しません。

---

## Mission Complete演出

Mission Complete時は

**色ではなくアニメーション**

で達成を表現してください。

現在のStatusカラーを維持したまま、

胸ランプから外側へ柔らかい光のリングが広がる演出を1回だけ再生します。

イメージ

・ランプが少し明るくなる

↓

・同色のリングがふわっと外側へ広がる

↓

・リングがフェードアウト

↓

・元のStatus点滅へ戻る

アニメーション時間は約1秒。

音は鳴らしません。

この演出は「任務を完了した」というイベントを表現するものであり、Statusの変更とは独立しています。

---

## ④ チャット

ロボタンが短く褒めます。

例

🤖
任務完了でござる。

🤖
よくできたでござる。

🤖
一歩進めたでござるな。

---

## ⑤ 次のMission生成

AIが次のMissionを生成してください。

---

## ⑥ Mission更新

Mission欄を新しいMissionへ更新します。

Tagsも更新してください。

---

## ⑦ Mission更新通知

チャットで一言だけ通知します。

例
```
🤖
新しい任務を設定したでござる。

🤖
次はこれを一緒にやるでござる。

🤖
焦らず一歩ずつでござる。

```

---

# Missionが存在しない場合

Mission欄は空にしません。

```
今日の様子を教えてください
```

を表示してください。

---

# Tags

Missionに合わせて

3個

表示してください。

固定表示は廃止します。


---

# Mission更新演出

Missionが変更されたら

チャット内で一言だけ宣言してください。

例

```
🤖
任務更新でござる。
```

```
🤖
新しい任務を設定したでござる。
```

```
🤖
焦らず、一歩ずつ進めるでござる。
```

Mission表示が切り替わるタイミングと同期してください。

---

# UI演出

Mission Complete表示中

・MISSION COMPLETE を表示（約1〜2秒）

・胸ランプを一度だけ柔らかく点灯

・音は鳴らさない

Mission更新後

・Mission欄を書き換える

・Tagsを書き換える

・チャットで任務更新を通知する

---

# 設計思想

Missionは

「今日の目標」

ではありません。

ロボタンが

**今、一緒に進める次の一歩**

です。

Missionを一つ終えたら、

ロボタンは必ずその努力を認め、

次の小さな任務を提案します。

この流れを繰り返すことで、

へなちょこが安心して一歩ずつ進める体験を目指してください。

Missionは雑談では更新せず、

行動や状況が変わったときのみ更新してください。
