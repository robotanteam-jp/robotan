# POWER / FUEL を会話から自動更新する仕様v2.0

v1.0では、POWER と FUEL は固定値になっていました。

ロボタンでは、会話内容から POWER と FUEL の変化量をAIが推定し、自動で更新します。

## 設計方針

POWER と FUEL は「へなちょこの現在の状態」です。

AIが現在値を直接決めるのではなく、

現在値に対する変化量（Δ）だけを返してください。

現在値はフロントエンド側で保持します。

---

## AIが返すフォーマット

JSONで返してください。

例

```json
{
  "powerChange": -15,
  "fuelChange": -5,
  "reason": "疲労が見られるため"
}
```

---

## 更新方法

現在

POWER = 72

FUEL = 60

AIが

```json
{
  "powerChange": -10,
  "fuelChange": -5
}
```

を返したら

POWER = 62

FUEL = 55

に更新してください。

## 制限

POWER と FUEL は

0〜100

の範囲に収めてください。

100を超えない。

0未満にならない。

---

## 注意

AIは現在値を決めません。

AIは変化量だけを返します。

現在値の保持と加算処理はフロントエンド側で行ってください。

このPOWER/FUELからSTATUSを自動判定します。
