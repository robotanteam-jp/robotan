# Spring（ゼンマイ）仕様v1.0

Springは状態を表すUIではありません。

Springはロボタンの非常用手動起動装置です。

通常時は静止しています。

色・数値・ゲージは持ちません。

普段はデザイン上のアクセントとして表示されます。

通信障害や災害など、

通常のAI機能が利用できない場合のみ、

手動ゼンマイ駆動へ切り替わる可能性があります。

通常の会話・Mission・Status・Power・Fuelとは独立しています。

## 表示

| 項目       | 内容                                  |
| -------- | ----------------------------------- |
| 表示       | **READY**                           |
| 色        | 🟠 オレンジ固定                           |
| 通常       | 静止（回らない）                            |
| 意味       | 非常用ゼンマイ待機中                          |
| Statusとは | **無関係**                             |
| Powerとは  | **無関係**                             |
| Fuelとは   | **無関係**                             |
| 将来       | オフライン時のみ `ENGAGED` または `MANUAL` に変化 |


ゼンマイは、ロボタンが最後までへなちょこのそばにいるための「最後の備え」です。
通常は使われることはありません。
